# Audio Visualizer

Next.js audio visualizer app with YouTube integration.

## Getting Started

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `bun run dev` — Next.js dev server
- `bun run build` — production build
- `bun run start` — start production server
- `bun run lint` — ESLint
